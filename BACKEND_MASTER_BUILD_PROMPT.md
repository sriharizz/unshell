## MCP SERVER — ENDPOINTS (port 8002)

POST /tools/fetch_uk_api — body: {crn: string}
POST /tools/query_ofac — body: {name: string}
GET  /tools/known_addresses — returns list of shell address strings

### fetch_uk_api Return Schema
company_name, crn, incorporation_date (YYYY-MM-DD), sic_codes, registered_address, raw_address_string, officers [{name, role, appointment_date, resignation_date, is_corporate}], pscs [{name, type, ownership_band, ownership_pct, jurisdiction, natures_of_control, is_offshore}], filing_count

### query_ofac Return Schema
match (bool), detail, program, matched_name

## TIERED AI ARCHITECTURE

### Node 2A — fetch_uk_api_node
1. Calls MCP fetch_uk_api → gets raw Companies House JSON
2. Passes raw JSON to ai/nvidia_normalizer.py
3. NVIDIA NIM (meta/llama-3.1-70b-instruct) via .with_structured_output() returns NormalizedGraph
4. Returns nodes and edges ready for graph engine

### Node 2B — extract_pdf_node
1. Passes pdf_bytes to ai/gemini_extractor.py
2. Gemini 2.5 Flash reads pages as images with generation_config response_mime_type=application/json
3. Returns OwnershipExtraction Pydantic model
4. Converts to nodes/edges and MERGES with existing state (additive)
5. All AI-extracted edges have trust_score=0.4

## GRAPH ENGINE — RULES (graph/engine.py)

Zero external calls. Pure functions. All values below are NON-NEGOTIABLE.

### Risk Vectors
AGED_SHELL: incorporation > 5 years AND filing_count < 3 → +15
VAGUE_SIC: sic in {74990, 99999, 74100, 82990, 70100, 74200} → +15
BOILER_ROOM: address in known_shell_addresses → +20
SMURF_NETWORK: 3+ PSCs with ownership_pct 15.0–24.9 → +25
CIRCULAR_LOOP: nx.simple_cycles() finds any cycle → +100 (FATAL)
NOMINEE_PUPPET: any node degree_centrality > 0.15 → +75 (FATAL)

Final score = min(raw_score, 100)

### Score Thresholds
0–29 → LOW_RISK → Auto-Approve
30–64 → MEDIUM_RISK → Human Review
65–94 → HIGH_RISK → Auto-Reject
95+ → CRITICAL → Auto-Reject + SAR Required

## LANGGRAPH ORCHESTRATOR — 7 NODES (agent/orchestrator.py)

input_router_node → routes by mode
fetch_uk_api_node → MCP + NVIDIA NIM → state update
extract_pdf_node → Gemini → merge into state
calculate_risk_node → graph/engine.py → state update
offshore_router_node → conditional routing
human_in_the_loop_node → interrupt() → paused state
sanctions_check_node → MCP query_ofac → state update
compile_output_node → builds final_payload → Section 8 schema

### Graph Wiring
START → input_router_node (Command routing)
fetch_uk_api_node → calculate_risk_node
extract_pdf_node → calculate_risk_node
calculate_risk_node → offshore_router_node (Command routing)
sanctions_check_node → compile_output_node
compile_output_node → END

Checkpointer: MemorySaver
interrupt_before: human_in_the_loop_node

### Public Functions
run_investigation(mode, crn, pdf_bytes, thread_id) → final_payload dict
resume_investigation(thread_id, pdf_bytes) → final_payload dict
